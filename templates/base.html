Acerca de

El propósito de este sitio, es de concentrar contenido relevante sobre la jornada electoral realizada en Cozumel durante el domingo 7 de julio del año 2013.

¿Cómo funciona?

Esta web application opera gracias a las APIs que twitter pone a disposición para que desarrolladores puedan hacer uso de su plataforma.

Son registrados únicamente los tweets que provienen de cuentas sin ninguna restricción de privacidad.

El proyecto no tiene ánimos de lucro, se elaboró con intención recreativa y educativa, por lo tanto, la información recabada en este lugar no pretende ser manipulada para otros fines.

Deslinde de responsabilidades

El contenido (tweets) mostrado en este sitio es propiedad de cada autor, el desarrollador no se hace responsable por las opiniones desplegadas y controversias que estas puedan causar.

Así mismo, este proyecto no está ligado a ningún partido político y tampoco busca dañar la imagen de ningún candidato.

El usuario es libre de generar su propio criterio sobre lo que cada tweet expone.
